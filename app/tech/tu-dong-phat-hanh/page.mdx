# Tự động phát hành phiên bản

Using semantic-release to fully automated version management and package publishing

## Cài đặt ban đầu:

Cài đặt package

```sh
yarn add semantic-release --dev
```

Thêm file cấu hình. Trong ví dụ này xài cú pháp module nên dùng đuôi `mjs`
ta có `release.config.mjs`

```mjs
/**
 * @type {import('semantic-release').GlobalConfig}
 */

export default {
  branches: [
    "+([0-9])?(.{+([0-9]),x}).x",
    "main",
    "next",
    "next-major",
    { name: "beta", prerelease: true },
    { name: "alpha", prerelease: true },
  ],
  plugins: [
    "@semantic-release/commit-analyzer",
    "@semantic-release/release-notes-generator",
    "@semantic-release/npm",
    "@semantic-release/github",
  ],
};
```

Theo lý thuyết thì chạy lệnh bên dưới là đã release được rồi.

```sh
npx semantic-release
```

Nhưng cần thêm một số cấu hình npm và ci/cd để tự động trên github action.
